[book] Static Site Generators

#books  #O'Reilly 

Заинтересовался я недавно созданием статических страничек на github.io. Начал что-то там ковырять, сделал HelloWorld и тут внезапно новость о том, что O’Reilly раздает бесплатно целую подборку книг, одна из которых Static Site Generators. Modern Tools for Static Website DevelopmentВ книге описаны современные генераторы статических сайтов, такие как Jekyll, Hugo, Wintersmith. В книге всего около 50 страниц, читается всё легко и на одном дыхании. Полезно для общего развития.Больше всего понравилась цитата: Static websites today are just like vinyl LPs: they’re coming back.
